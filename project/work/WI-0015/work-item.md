# WI-0015: Left-To-Right Subtitle Attribution Typing

## Summary

Make the homepage banner author/source line type left-to-right while remaining right-aligned to the subtitle block.

## Acceptance Criteria

1. Author/source line remains visually right-aligned after typing completes.
2. During typing, author/source text grows left-to-right instead of pushing left from the right edge.
3. Typewriter cursor stays after the currently typed author/source text.
4. `npm run build` passes.
