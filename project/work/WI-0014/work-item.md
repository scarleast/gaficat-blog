# WI-0014: Dynamic Subtitle Attribution Alignment

## Summary

Align the homepage banner subtitle attribution line to the rendered subtitle text width instead of a fixed subtitle container width.

## Acceptance Criteria

1. Author/source line right edge follows the subtitle text block width.
2. Short subtitle lines no longer push the author/source line to a far fixed right edge.
3. Typewriter cursor still follows typed text.
4. `npm run build` passes.
