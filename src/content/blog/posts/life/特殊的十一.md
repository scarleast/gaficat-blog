国庆的几天，结束了三年的租房生活，父母来我的城市帮我搬家，当然了，也不忍心父母7天完全成了我的保姆，也安排了几场演出带父母去看，我们一家子都觉得，这应该是近几年最有意义的一个国庆了。

# 国庆的劳逸结合

## 永远的邓丽君-弦乐钢琴五重奏

<center>    <img style="border-radius: 0.3125em;    box-shadow: 0 2px 4px 0 rgba(34,36,38,.12),0 2px 10px 0 rgba(34,36,38,.08);zoom:20%;"     src="https://pic.gaficat.com/default/永远的邓丽君-弦乐及钢琴五重奏纪念票.jpg">    <br>    <div style="color:orange; border-bottom: 1px solid #d9d9d9;    display: inline-block;    color: #999;    padding: 2px;">永远的邓丽君-弦乐及钢琴五重奏纪念票</div></br> </center>

父亲是邓丽君的歌迷，刚好看到XSO交响乐团在国庆有邓丽君的弦乐钢琴五重奏，就是下面这首《我只在乎你》的小段，感觉太美了，朦朦胧胧听到了《落叶归根》的旋律，也听到了电视剧《激情燃烧的岁月》片头片尾曲的影子，索性订了三张票。


现场听弦乐钢琴的听感和耳机里确实不一样，你能听到弓摩擦弦发出奇妙的泛音，你能听到斯坦威钢琴金钱的力量，演出的第二首曲子《在水一方》，听的人一直在起鸡皮疙瘩。早些年，音乐制作水平有限，所以我不太喜欢早期唱片编曲给人的感觉。而改编的纯器乐演奏没有了人声，旋律完全由弦乐和钢琴烘托，人的注意力就全部集中到了旋律上。我虽然不是邓丽君的歌迷，但是发现邓丽君的歌旋律写的太棒了，离调和转调不要太巧妙。


整个国庆假期看的几场演出，我最喜欢的就是这场，一方面邓丽君的歌，曲子写的太唯美了，另一方面由人唱的歌曲改编、扩充成器乐，对作曲家的要求很高。演出间隙，我一直在跟母亲说，真的很好奇这场演出幕后的编曲是哪位老师，西安交响乐团由这么厉害的作曲家吗？我太喜欢这个改编了。后来，查到了本场演出的曲目其实来自瑞鸣《永恒的魅力》这张专辑，音乐制作人叶云川去纽约，召集了一批新锐音乐家，以古典五重奏的形式制作了这张专辑。担任编曲的，是来自台湾，曾于纽约州立大学获得作曲、钢琴专业双博士学位的杨岚茵。

当然了，改编很棒，演奏家们的水平也很高超，也听过其他的弦乐现场，但是XSO给人的感觉不是演奏者纯粹盲目拉对音就完事大吉。节奏的急缓、力度的轻重、配合的默契都完全挑不出来什么毛病，很棒的一场演出。

## 让玫瑰带去幸运-中外艺术歌曲专场

<center>    <img style="border-radius: 0.3125em;    box-shadow: 0 2px 4px 0 rgba(34,36,38,.12),0 2px 10px 0 rgba(34,36,38,.08);zoom:20%;"     src="https://pic.gaficat.com/default/让玫瑰带去幸运-中外艺术歌曲专场纪念票.jpg">    <br>    <div style="color:orange; border-bottom: 1px solid #d9d9d9;    display: inline-block;    color: #999;    padding: 2px;">让玫瑰带去幸运-中外艺术歌曲专场纪念票</div></br> </center>

母亲是声乐演员出身，所以选了这场演出，演出的大部分作品我都没有听过，不过还是有几首熟悉的，比如邓垚老师的徒弟，王龙作曲的《锦瑟》。久石让的《世界的约定》这次演出是钢琴、单簧管二重奏，单簧管的低音太轻了，不知道是乐器本身的问题还是演奏者的问题，没有江老师和Eric的钢琴、长笛版本好听。

本场的意外收获，是听到了《圣母颂》，就是下面这段。我觉得韩国流行音乐厉害的地方在于，融入了很多古典的元素进去。比如这段《圣母颂》，加入了打击乐和弦乐，让本来节奏舒缓的歌曲，变得多了几分动感和紧张感。我一直以为这首歌是原创的，没想到是改编的古典音乐。

## 青春收藏馆-流行歌曲合唱专场

<center>    <img style="border-radius: 0.3125em;    box-shadow: 0 2px 4px 0 rgba(34,36,38,.12),0 2px 10px 0 rgba(34,36,38,.08);zoom:20%;"     src="https://pic.gaficat.com/default/青春收藏馆-流行音乐合唱专场纪念票.jpg">    <br>    <div style="color:orange; border-bottom: 1px solid #d9d9d9;    display: inline-block;    color: #999;    padding: 2px;">青春收藏馆-流行音乐合唱专场纪念票</div></br> </center>

这场演出按照指挥的说法，是一场温暖的演出，曲目都是一些耳熟能详的作品，有TFBOYS的《青春修炼手册》、有邓丽君的《我只在乎你》、也有灌篮高手的主题曲。有意思的是，《让玫瑰带去幸运-中外艺术歌曲专场》的独唱演员这次到了合唱的队伍中，虽然服装混入了队伍中，不过美声演员魁梧的身材，还是一眼就能认出来。

这场音乐会的感受是，日本的歌曲旋律听起来气场不足，按照方言说“窄溜儿”的国家。国家太小了，文化所致，音乐很难写的很大气。而我最喜欢的歌，是《父亲写的散文诗》，指挥在这首歌开始之前拿起话筒：“为什么在这个不前不后的时间拿起话筒呢？因为我们有太多写母亲的歌了，而写父亲的歌屈指可数。接下来这首歌，是我们交响乐团的驻团作曲家（我忘记名字了）改编的，我们希望把这首歌送给在场的所有父亲，以及在场各位的父亲。”现场的合唱版本比任何独唱版本更打动人，一首歌唱完，演员和观众，所有人眼眶都是湿漉漉的。我也终于理解了母亲一直想组建合唱团的原因，人嗓子是最好听的乐器，优秀的合唱，没去现场听过，无法感受到其魅力。


最后，演出结束，指挥和钢伴收到了观众的献花，指挥说：“我在舞台上指过挥，唱过歌，唯独没有收到过花。”紧接着没有防备，一把将献花抛向了观众，刚好被我们接住。

# 异乡打拼的坚守

17年毕业季，我四处投递简历，最终决定到家乡所在省会工作，虽然学生时代积攒下来一种不愿被管束的情绪，但家庭教育的关系，终究还是不想离家太远。今年十一，新房除了基本的电器，任何家具都没有，但我还是急着想搬进新家。在外租房，虽然说房子是别人的，生活是自己的，但总感觉会搬家，静不下心，一室一厅通风不好的鸽子笼里面，生活没什么动力。

最近有一件事感触挺深的，自己一个人在异乡，装修收尾的半个月，大小所有事都是一个人盯着，虽然累，但是感觉也成长了不少。像电线多粗的能带动多大的电器，热水器天猫安排的安装师傅不是品牌官方的，来了几次一直漏水怎么解决等等。生活里的琐碎，小时候是父母操心，租房的时候是房东操心，有了自己的房子，就都是自己的事儿。突然明白了一件事，我自己父母也跟我说过：“你要是能找个本地的姑娘就好了，能给你们帮上一些忙，你们能轻松些。”我一直不以为然，自己在一个新城市打拼有多难？倒退一代人，大家不都是从农村出来，在城市里打拼吗？父母都希望子女能过的轻松些都能理解，一个人打拼的苦，只有自己经历过才知道。但是跟房东退房的时候，房东的母亲说：“没办法呀，你舍不得放手，他们永远都是长不大的孩子。”

那你说，为什么还要到大城市打拼呢？我跟母亲说：“你看，刚才那场演出，后面那位父亲带着自己小女儿来看音乐会，小孩子从小就能接触到这些东西，这么棒的改编，潜移默化会影响她的音乐审美的。我们家的那些亲戚们呢？啊怡喜欢美术，想看个美术展，没办法，小地方没有那么多美术展给你看，尽管舅舅开一百万的车。啊玉想听个音乐会，也没办法，小地方没有那么多音乐会给你看。这就是为什么我想留在大城市，尽管对我来说，对咱们家庭来说，回到小城市，就完全不会有任何生活的压力了，我们活的会相当轻松。但，很多时候不光为了自己呀！也许我没办法给我的孩子大城市最好的教育资源，但是对我来说，我的小孩子，平时带他去听听水平高的音乐会，哪怕就是遇上今天商场里领克给小朋友准备的安全展，都很棒，起码这个城市有这样的资源，我回去，哪会有这样的资源？大城市的孩子，赢在了起跑线上，即使不求他成才，这些大城市的资源，能给他一个快乐的童年啊！”

一个人折腾装修，实在是筋疲力竭了，尽管父母说：“哪个姑娘找到你可真是享福啊，你太细致了，洗碗机、电动晾衣架，所有的灯等等，家里啥东西你都自己安排好了。”但搬家、打扫卫生对我来说，一个人还是太吃力了。父母趁着国庆，开车来帮我搬家，几乎每天都忙到半夜十二点。搬家我的唯一印象是，母亲7天时间，除了吃饭、睡觉，没有一刻是坐下来休息的。

> 搬家时，我约了一辆货拉拉，一次把家里打包好的大件运到新房，货拉拉搬家的小哥倒是手脚利索，非常乐观。我跟他说很不理解父母，跟门卫说下，把车开进小区，没拉下的一次装车上拉走就行了啊，非要两个人抱着东西去停车场。父亲年纪大了，反应慢了，视力差了，开车不行了。货拉拉小哥说：“人年纪大了都正常，都怕麻烦，跟门卫说他们也说不清楚，所以就自己抱着走。年纪大了没办法，尽量让他们晚上别开车吧，尤其是晚上，他们都看不清楚。”我猛地意识到，父亲无意间说：“我看手机导航都看不清楚，得扶半天才行。”我小时候，眼睛近视，没有配眼镜的时候，确实也感觉整个人感官都慢了。我跟母亲说：“你回去带我爸配副眼镜吧，就在我当时配眼镜的地方，验光师还可以，我这副眼镜带了5年了，没感觉看东西眼睛疲劳，前几天去医院复查视力，矫正视力能看到1.2，医生都挺诧异的。人看不清东西，反应什么的跟着就慢了，你想想你如果不戴眼镜，看不清东西，多难受哦，你再想想我爸，导航都看不清楚了。”

这次搬家，父亲一直在说，以后还是要多出来看看，不然就成土包子了。确实，我刚到新的城市，停车场空车位的灯光表示，自助结账的购物台，都没见过，也都是慌慌张张的。

> 我心想父母来帮我搬家，好说歹说得带父母去吃顿海底捞吧？结果节假日，车流量太多，商场停车场排队的通道已经被交警用锥桶延伸到商场外的马路上，不熟悉路线，父亲只好插着锥桶的空隙变道，后方车辆看我们变道也跟着变道，造成了拥堵。交警过来拍窗子，很生气说：“我也不让你们倒出去了，你们自己下来给我把锥桶重新摆好！”父亲见状，跟小孩子做了错事一样，一下子不知道该前进还是后退，或是下车摆锥桶。我立刻下车，想跟交警沟通，看怎么解决，结果一只脚刚落地，父亲一脚油门，车轮随即一点一点吞食我的后脚跟，直到脚腕的皮被车轮蹭掉了。我一瘸一拐的摆好锥桶，跟交警说，我们是外地车，不熟悉路线，不好意思。心里五谷杂陈，一方面是城市发展太快了，别说乡村了，小城市的人都跟不上了。另一方面，父母老了，衰老的速度我追不上，他们越活越像个怕惹事的小孩子了。那能怎么办呢？我得扛起单子，我来解决问题呗。未来有了妻子，四个老人，不都指望着这一个男人呢，尽可能快速成熟起来吧。

父母来帮忙搬家的几天，每天上午起床，杯子里有倒好的热水和准备好的早餐，起来就能看到父母，好像就几天时间就完全习惯了。父母回去的时候，我拍下了这张照片。我的两个保温杯倒满了热水，中间是昨天音乐会我们接到的玫瑰花，偌大的房子，就我一个人，冷冷清清。
